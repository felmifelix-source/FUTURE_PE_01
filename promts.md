# Prompt Engineering Documentation

## Prompt Used

Act as a professional website copywriter.

Create website content for a modern café called Brew Haven Café.

Target audience:

- Students

- Professionals

- Families

Include:

1. Homepage hero section

2. About section

3. Why choose us

4. Services page

5. Multiple call-to-action sections

Tone:

Friendly, modern, welcoming, and conversion-focused.

The content should be website-ready and easy to read.

---

## Prompt Logic

- Clearly defines the business type (café).

- Specifies the target audience.

- Requests structured website sections.

- Defines the desired tone.

- Produces content suitable for a real business website.

## Expected Outcome

Generate persuasive website copy that can be directly used on a café website.
